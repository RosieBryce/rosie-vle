Activation Functions

Activation functions introduce non-linearity into neural networks — without them, stacking linear layers is equivalent to a single linear transformation. Each function has different saturation behaviour and gradient properties that affect how well a network trains.

Options
FunctionFormulaOutput rangeGradient issueWhen to use
Sigmoid 1 / (1 + e⁻ˣ) (0, 1) Saturates for |x| > 4; gradient → 0 (vanishing) Output layer for binary classification only
Tanh (eˣ − e⁻ˣ) / (eˣ + e⁻ˣ) (−1, 1) Saturates for |x| > 2; stronger gradient than sigmoid Hidden layers (better than sigmoid — zero-centred)
ReLU max(0, x) [0, ∞) Dead neuron: gradient = 0 for x ≤ 0 (neuron never recovers) Default for hidden layers in CNNs/MLPs — fast, sparse
Leaky ReLU x > 0 ? x : 0.01x (−∞, ∞) Small gradient for x < 0 (no dead neurons) Drop-in ReLU replacement when dead neurons are a concern
ELU x > 0 ? x : α(eˣ−1) (−α, ∞) Smooth near zero; can produce negative outputs → zero mean When you need smooth gradients everywhere; α usually = 1
What to Notice
  • Sigmoid and tanh saturate: Turn on saturation zones and see the flat regions at the extremes. The derivative (f′) is near zero in those zones — this is the vanishing gradient problem. Gradients from these neurons barely update earlier layers.
  • ReLU derivative: Toggle f′(x). ReLU is 0 for x ≤ 0 (exactly zero, not approximately). Any neuron receiving consistently negative input is permanently dead — its gradient is zero, so it never updates. Leaky ReLU fixes this with a small slope.
  • Tanh is better than sigmoid for hidden layers: Its output is zero-centred (−1, 1) vs (0, 1). Non-zero-centred outputs cause gradient updates that are all positive or all negative, slowing convergence.
  • ELU's smooth negative region: Unlike ReLU, ELU outputs negative values and is differentiable everywhere. This can lead to zero-mean activations naturally, which helps with gradient flow.
  • For the assessment: Know which activation saturates (sigmoid, tanh) and which doesn't (ReLU, Leaky ReLU). Know that ReLU is the standard default for hidden layers and sigmoid is for binary output layers.
